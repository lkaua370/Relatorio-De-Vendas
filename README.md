# 📊 Sistema Relatório de Vendas

Um sistema web completo para o gerenciamento de inventário, registro de vendas e acompanhamento financeiro, construído com **Python** (Flask) no back-end e **HTML/CSS/JS** modernos no front-end. O projeto se destaca por uma interface visual imersiva e responsiva em tons escuros e *glassmorphism*.

---

## 🚀 Principais Funcionalidades

- **Controle de Inventário:** Cadastro e listagem de produtos com categorias.
- **Gestão de Vendas:** Lançamento de vendas associando clientes a produtos, com cálculos automáticos.
- **Dashboards Visuals:** Gráficos dinâmicos (Chart.js) para análise de métricas financeiras (Faturamento, vendas por categoria).
- **Segurança:** Autenticação de usuários com sessões bloqueadas para visitantes.

---

## 🛠️ Stack Tecnológica

- **Back-End:** Python 3, Flask, SQLite3 (banco de dados)
- **Front-End:** HTML5, CSS3, JavaScript Vanilla, Chart.js
- **Segurança:** Werkzeug Security (hashes de senhas)
