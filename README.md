# Relatório de Vendas - Klaus

Um sistema web completo para gerenciamento e controle de vendas, desenvolvido com **Python**, **Flask** e **SQLite**. 

O projeto conta com interface responsiva e moderna, focada na experiência do usuário para facilitar o cadastro de produtos, registro de vendas, autenticação e visualização de relatórios.

## Funcionalidades
- Cadastro e gerenciamento de Produtos
- Registro e detalhamento de Vendas
- Relatórios visuais (Dashboards)
- Autenticação de Usuários
- Interface futurista e interativa
